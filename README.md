# Helpline Welfare Trust — NGO Website

A modern, responsive NGO website built with **Next.js**, **Tailwind CSS**, and **Framer Motion**. Designed for Helpline Welfare Trust (Pakistan) with an Islamic/welfare theme (green, white, light gold), donation focus, and placeholder areas for images.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Pages

- **Home** — Hero, Who We Are, What We Do, Impact stats, CTA
- **Who We Are** — About, values, main branch
- **What We Do** — Services (Education, Clean Water, Healthcare, etc.)
- **Our Impact** — Stats and impact story placeholders
- **Campaigns** — Water Pump, School Support, Ramadan Ration, Disaster Relief
- **Education Projects** — Intro, Al-Kitab highlight, schools list (cards)
- **Clean Water** — 2000+ pumps, provinces, impact, solutions
- **Our Presence** — Cities (Lahore, Karachi, Hyderabad, Gojra, Bahawalpur, Abbottabad) + map placeholder
- **Sponsors / Supporters** — Logo carousel placeholders
- **Donate Now** — One-time/monthly, amount, Bank Transfer / Online Payment
- **Contact** — Form (Name, Email, Message), address, phone/email/map placeholders

## Design

- **Colors:** Primary green (teal), white, accent gold (amber)
- **Placeholders:** Image areas use `ImagePlaceholder`; replace with real images when ready
- **Animations:** Framer Motion on sections and cards
- **Responsive:** Mobile, tablet, desktop

## Add your content

1. Replace image placeholders with real photos (hero, about, schools, water, impact, campaigns).
2. Add phone and email on the Contact page.
3. Embed Google Map on Contact (Rana Town, Shahdara, Lahore).
4. Add bank details and payment gateway on Donate page.
5. Add sponsor logos on Sponsors page.
6. Optional: add Pakistan map with markers on Our Presence.

## Build

```bash
npm run build
npm start
```
